## ✨ Overview

**EchoTune** is a high-performance, dynamic music streaming web application built with a premium desktop-first approach. Powered by the **DeepSeek AI Radio** engine, EchoTune learns your music taste and curates infinite, highly accurate playlists based on any mood, genre, or activity you feed it.

Say goodbye to manual queuing and hello to intelligent, high-fidelity (320kbps) seamless streams.

## 🚀 Key Features

* **🤖 AI Chat Music Discovery** — Describe your vibe (e.g., "Late night coding session" or "Upbeat 80s synth-wave") and the AI curates the perfect playlist instantly.
* **📻 DeepSeek Radio Engine** — An intelligent recommendation system that cross-references accurate song choices to map them seamlessly into the active queue.
* **🎵 High-Fidelity Audio** — Built-in support for 320kbps JioSaavn streams for a top-tier listening experience.
* **🎨 Modern UI/UX** — A hyper-responsive, beautiful interface featuring dynamic themes, smooth interactions, and a polished Library.
* **⚡ Blazing Fast** — Powered by Vite + React 18 for instantaneous loads and smooth client-side routing.

## 🛠 Tech Stack

* **Frontend:** React 18
* **Build Tool:** Vite
* **AI Integration:** DeepSeek LLM API 
* **Styling/Components:** Custom CSS & Component Library

## 💻 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/echotune-web.git
   cd echotune-web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/`.  
   *(To test on your mobile device, run `npm run dev -- --host` and visit the generated Network URL on your phone).*

## 🌍 Deployment

Deploying EchoTune is incredibly straightforward. We recommend using **Vercel** for a zero-configuration deployment:

1. Push your code to your GitHub repository.
2. Log into [Vercel](https://vercel.com/) with GitHub.
3. Import the `echotune-web` repository.
4. Click **Deploy**. Vercel will automatically detect Vite and handle the rest!

*Alternatively, it can also be deployed to **GitHub Pages** by specifying the repository base path in `vite.config.js`.*
